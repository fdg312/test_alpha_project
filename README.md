## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Quick Start

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.
